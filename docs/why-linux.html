2.1 Why linux?

When working on large projects and big datasets you will likely encounter distrbuted computing running onunix like system which usually means a linux distribution or a version MacOS. Cloud computing services like Amazon’s EC2, Google’s Compute Engine work on linux Virtual Machines so you will have to have some linux knowledge to leverage them. Most importantly, when working with a technology you are not familiar with you will likely stumble upon a post, tutorial or a course which will most likely use linux/OSX.

Also it is worth mentioning that most linux distributions(variaties of linux operating system) are free to use. On top of the a lot of free, open-source is available to linux users.

Last but not least since linux is open-source it is highly customizable to your needs.

This chapter will get you acquainted with linux environment if ou are on a Mac you should be able to follow along with minimal code changes.
