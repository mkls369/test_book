2.1 Getting acquainted with the command line, part 1

If you are running linux you should be able to search for ‘terminal’ to start a terminal session. If you are working on Windows an easy way to emulate an Ubuntu terminal is by installing Windows Subsystem for Linux.

First let’s get somw terminology out of the way you will often hear these words when referring to the black (or dark colored) screen with a blinking cursor: terminal(emulator), console, command line, shell. Below are definitions from The Linux Information Project

Terminal - A terminal window, also referred to as a terminal emulator, is a text-only window in a graphical user interface (GUI) that emulates a console.

The original meaning of console as it related to computers was (1) an instrument panel containing the controls for a computer … Another term for console as used in this sense is terminal.

These two terms have historical origins - you can read about them in their respective linfo pages in our case they are often used interchangeably to refer to program that emulates a machine that is used to input data.

A command line is the space to the right of the command prompt on an all-text display mode on a computer monitor (usually a CRT or LCD panel) in which a user enters commands and data.

So the command line is the textual interface - a place for you to input commands for the computer to execute.

A shell is a program that provides the traditional, text-only user interface for Linux and other Unix-like operating systems. Its primary function is to read commands … and then execute (i.e., run) them.

Finally shell is the program that allows for easy human-computer communication - it relays commands to the deeper layers of the OS. In later Sections we will introduce commands that are executed by the shell. One important thing to note is that shells are extremely customizable, we will be using bash because of its popularity and availability.

We will be introducing a more terms along this book, it’s a good habit of looking them up to avoid confusion.
